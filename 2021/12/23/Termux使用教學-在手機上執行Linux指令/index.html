Termux使用教學: 在手機上執行Linux指令

發表於 2021-12-23 更新於 2021-12-24 分類於 電腦科學 Disqus：

Android作為類Linux系統，卻沒有終端機。而Termux這類終端機模擬器補全了這個缺口，不需要Root就可以使用。

為什麼要在手機上敲指令? 電腦很多東西都透過命令列運作，有的時候手機並沒有對應的APP，你還是可以用終端機來執行操作。舉例來說，Termux可以:

  • 編譯執行C/C++、Java、Python、NodeJS程式
  • 文字介面遠端ssh到主機
  • 手機架設Apache伺服器
  • 執行QEMU虛擬機
  • 執行docker
  • 安裝Proot Ubuntu等Linux發行版 + VNC Server
  • 破解電信限速

使用Termux還有一個好處，那就是讓你熟悉各種Linux指令，基本的ls都能夠使用。當然要控制Android系統還是需要Root權限。

安裝Termux

  • F-Droid下載
  • Github
  • Google Play

儘管Google Play有上架，但是更新很慢問題又多，因此建議到F-Droid這個開源商店下載。

點進F-Droid的連結，點選Download APK下載安裝檔。

Termux初始設定

  1. 開啟後就是終端機介面，點一下輸入框鍵盤會自動開啟，螢幕上已有一些鍵盤快捷鍵。

  2. 從通知欄下拉，按Exit會終止Termux，Aquire Wakelock則是讓其保持在背景執行。

  3. 從螢幕左側滑進來，按New Session開啟多個終端機，Keyboard喚醒鍵盤。

  4. 現在可以開始打第一個指令了，為了讓Termux可以存取手機的檔案，必須先設定儲存空間權限，之後同意權限 (在Termux空白處長按可貼上複製的指令)

  6. 更新時可能會問要不要升級，一律輸入y。之後就能開始安裝常用套件了。

Termux相關軟體

在F-Droid商店還有其他Termux相關的插件軟體。

  • Termux:API: 讓Termux使用更多手機功能，例如傳送簡訊

  • Termux:Boot: 在開機時執行自訂指令

  • Termux:Float: 以懸浮視窗執行Termux

  • Termux:GUI: 使用Python資料繫結撰寫GUI

  • Termux:Styling: 自訂Termux終端機樣式

  • Termux:Tasker: 快速啟動指定任務

  • Termux:Widget: Termux的桌面小工具

  • Termux:X11: 用X11協定顯示圖形介面的插件
