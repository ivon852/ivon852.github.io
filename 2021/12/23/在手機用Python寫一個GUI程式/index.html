在手機用Python + Termux寫一個GUI程式

發表於 2021-12-23 更新於 2021-12-24 分類於 電腦科學

Termux透過插件Termux:GUI，呼叫Android元件來顯示圖形介面。
可使用Python或C語言的data-binding寫GUI，作者提供了很多方法，用簡短的程式碼就能建立介面。

安裝

Termux:GUI雖然已在F-droid上架，但目前0.1.3版有簽名問題。若無法安裝則必須解除安裝Termux，改從Github下載Termux，再來安裝Termux: GUI。

  1. 從F-droid或Github下載安裝Termux:GUI。

  2. 開啟Termux，安裝python-binding。
