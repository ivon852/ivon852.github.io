前言

GCC編譯時會跳出很多紫色和藍色的警告(Warning)，但那不是最重要的，重要的是紅色的錯誤(Error)，一遇到錯誤編譯器就會停下來，要將其解決後再make，繼續編譯。

include: no such file or directory

這是最簡單的錯誤，如果同一個目錄確實有該檔案，把include後面的<>改成" "，讓編譯器從同一個目錄去找檔案就可以了。
