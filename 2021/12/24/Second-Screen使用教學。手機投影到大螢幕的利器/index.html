Android手機當電腦用，實作與心得

發表於 2021-12-24 分類於 電腦科學 Disqus：

Android手機 + Type C Dock + 螢幕 = 簡單文書電腦

本文不使用Android 10實驗性桌面模式，也不是三星的Dex技術。需要Root權限。

需要的硬體

  • 手機：Sony Xperia 5 II，Android 11，支援Type-C HDMI螢幕畫面輸出。如果不支援可以用Chromecast無線投影。
  • Type C Dock，就是給筆電用的擴充插座，上面有USB、HDMI、電源的接口，如上圖。

建議裝置都以有線為主，當然藍牙鍵鼠也可以。

需要的軟體

  • Second Screen: 需要Root權限。調整投影到大螢幕上的解析度和DPI，因為手機21：9畫面在電腦上會有黑邊，或是按鈕太大。
  • GCIN: 實體鍵盤中文輸入法，Android系統到開發人員選項開啟「將活動強制設為可調整大小」、「啟用自由形式視窗」、「啟用自由形式大小」。不要開啟「強制使用桌面模式」。

軟體使用情況

  1. 在Second Screen設定欲輸出的解析度和DPI，按下Load Monitor。投影的是22吋16：9的螢幕。開啟後將一切硬體都插上。

  2. Word會自動切換版面，功能沒有變化。

  3. Second Screen有強制Chrome開啟桌面版網站的選項。沒有Root亦可下載Kiwi Browser達到同樣效果。

  4. 在多工畫面點選圖標可讓App進入自由形式，但是同一個app仍無法開多個「視窗」。

  5. Minecraft基岩版，接上實體鍵盤後，把輸入法調成英文的就能無縫切換到電腦版操作。

  6. Linux Deploy就比較有桌面系統的感覺，但chroot問題也很多。

如何切換回手機模式

關閉Second Screen，如果有開啟鎖定旋轉螢幕，則把Second Screen強制停止後就能恢復正常。接著再把Type C Dock拔掉。

很像iPad OS，但是細節還是不行，是可以做輕便工作。輸入法還是很難用。

純靠鍵鼠操作可能還要開Talkback，不然有些地方還是要用到觸控螢幕。
