QEMU安裝Windows 10，開啟/關閉顯示卡直通

發表於 2021-12-24 分類於 電腦科學

Linux顯示卡直通(passthrough)給Windows虛擬機，用來執行3D應用程式。

  • Host OS: Ubuntu 20.03

  • Guest OS: Windows 10

  • CPU: Intel i5-7400K (HD630)

  • GPU: GTX 1050Ti

建議要準備二個螢幕，或者Host OS的內外顯都要開啟。因為停用Host OS的顯示卡驅動重開機後，顯示卡會無法輸出畫面。

如果只有一個螢幕，則透過BIOS設定內顯優先，開機後線插在內顯的孔。

設定好之後，Windows虛擬機的裝置管理員應該會看到直通進去的顯示卡。

流程：

  1. 安裝QEMU，安裝Windows 10要測試的遊戲或程式。

  2. 內顯和外顯開啟。

  3. 設定IOMMU，以及輸入顯示卡的PCI 編號、vendor ID 與 device ID， 開啟vfio。

  4. 在Host OS把顯示卡的驅動程式黑名單，重開機。

  5. 在QEMU設定，把顯示卡直通(passthrough)給虛擬機Windows 10。

  6. 在Guest OS的Windows 10安裝Nvidia GeForce Experience驅動程式，就能玩3D遊戲。

參考這二篇文章：

  • KVM/QEMU 虛擬機器設定 GPU Passthrough 記錄

  • 閒聊 教學：如何在Linux上玩LOL

解決QEMU自動縮放：在Windows中安裝SPICE Guest Tools，這樣Windows的畫面才會隨虛擬機視窗縮放。

如果要取消顯示卡直通，把上面的流程倒著做，取消停用nouvea驅動，關閉IOMMU，更新GRUB後重開機，螢幕線就可以插回顯示卡的孔。
