Andronix: 安裝功能齊全的 Proot Ubuntu

發表於 2021-12-24 分類於 電腦科學

Andronix其實是一套輔助軟體，透過引導式的介面，讓使用者選擇想要安裝的Linux發行版，再將它們提供的指令或一鍵安裝腳本複製貼到Termux上執行即可，主打開箱即用。

Termux自行安裝Proot發行版總是碰到各種問題？Andronix提供懶人安裝的發行版系統，一鍵搞定音效、圖形的設定。如果要追求更好的品質也可以購買調校好的版本。

下載安裝

  1. 下載Andronix。該APP有圖形化介面指引用戶安裝Proot發行版，選取需要的選項後，再將其產生的指令貼到Termux，就會自動執行腳本，安裝對應的發行版+桌面環境。

  2. 除了免費腳本外，Andronix亦販售修改過的Linux發行版（Modded OS），花約100塊台幣購買，安裝腳本會自動解決音效輸出和圖像問題，附帶一個漂亮的桌面。需要註冊Andronix帳號才可以購買。

  3. 購買畫面，雖然用Google Play付款，但是購買的授權是跟著Andronix的帳號。

  4. 按Proceed之後會得到一組網址和指令，將其貼在Termux就會自動下載安裝。購買的是Ubuntu 20.04 + XFCE。

  6. 設定時區、鍵盤配置、建立使用者帳號，以後也是用這個腳本開機。

  7. 之後再用指令啟動Vncserver，就可用VNC Viewer進入桌面:

  8. Proot只能登出，在Termux輸入logout指令關閉Proot。

使用體驗

  1. XFCE桌面有深度客製化，預裝軟體有Chromium和Firefox二個瀏覽器。

  2. 還有LibreOfficeGIMP、VS Code OSS，其他視窗程式如blender也能正常開啟。

  3. 支援Pulse Audio音效輸出和Open GL(無硬體加速)

  4. 安裝中文字體（教育部標準宋體）
