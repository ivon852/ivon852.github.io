AltStore安裝教學

發表於 2021-12-06 分類於 電腦科學

AltStore可在iOS裝置安裝未經蘋果審核的第三方APP，例如去廣告的Youtube APP 「Cercube」。

不需越獄，任何型號的iPhone/iPad皆適用，每7天需要「重簽」，

該程式支援macOS / Windows 10。

安裝AltStore到裝置

  1. 電腦必須安裝iTunes和iCloud，不要從微軟商店下載。將iOS裝置插上電腦，點選信任此電腦，確認iTunes認得到iOS裝置。

  2. 下載AltStore (https://altstore.io/) ，解壓縮點選AltInstaller.msi安裝到電腦。從開始近安裝的程式找到AltStore，啟動之後會在右下角的系統匣顯示菱形圖示，並在背景執行AltServer。

  3. 按一下AltStore的菱形圖示，按Install AltStore，安裝AltStore到iOS裝置。

  4. 此時會要求登入Apple ID，這是為了註冊開發人員帳號。如果安裝出現錯誤訊息，可移至本文底部查看一些解決方法。

  5. 稍後裝置桌面應該就會出現AltStore的圖示，先不要開啟。到系統設定→一般→VPN與裝置管理，信任AltStore的開發者APP，之後就可以開啟AltStore。

安裝APP

安裝APP時建議與電腦上的AltServer保持連線。

要安裝APP必須要有該APP的IPA檔案，可透過Safari下載IPA檔案後，再從My Apps畫面點選左上角+號，選取IPA檔案，接著登入Apple ID，就會安裝APP。

如何重簽

  1. 在已安裝的APP列表會看到APP離需重簽還剩多少天，過期之後APP就會無法啟動。

  2. 將裝置連接到電腦，點選Refresh All即可重簽。也可以讓裝置與電腦處於同一個網路下，用Wifi重簽。

簡單除錯

  • 錯誤訊息: Invalid anisette data. Please close iTunes and iCloud then try again.: 刪除C:\ProgramData\Apple Computer\iTunes\adi資料夾，重新開啟iTunes，再用工作管理員強制停止。AltSever按下Close再重新開啟。

  • 錯誤訊息: Failed to verify code signature of: 請將Windows系統時間調整至正確的時間。
