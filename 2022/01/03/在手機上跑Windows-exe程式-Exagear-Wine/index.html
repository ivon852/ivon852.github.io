在手機上跑Windows exe程式 (Exagear + Wine)

發表於 2022-01-03 分類於 電腦科學 Disqus：

幾年前俄羅斯公司ElTechs發表的Exagear，能夠將X86翻譯成ARM的指令，配合Wine就能在手機執行電腦程式，不需要root。(Wine是讓Linux電腦能夠執行Windows exe的軟體)

雖然後來Exagear的技術被華為買走(相關討論)，不過Github有俄羅斯人釋出內含Exagear的Termux Proot版。主要原理是在Termux安裝32位元的Debian Proot環境，再用Wine執行32位元的Windows exe。

測試能跑的程式:

遊戲Fate/stay night

foobar2000

Notepad++

HoneyView

Chrome可攜版

安裝教學

  • 手機: Sony Xperia 5 II，Android 11
  • 需要APP: Termux (使用教學)、VNC Viewer

  1. 開啟Termux安裝套件

  2. clone這個repo，並加入submodule

  3. 下載解壓縮Debian 10檔案系統

  4. 使用腳本登入系統(加old參數是因為有時候登入會無法開啟終端機)

  5. 這樣32位元的Debian環境就設定好了。首先進行更新

  6. 安裝XFCE4桌面環境和常用工具

  7. 如果APT安裝出現E: Sub-process /usr/bin/dpkg returned an error code (1)，執行:

  8. 安裝Wine

  9. 啟動Vncserver，第一次啟動會要求設定六位數密碼。

  10. 開啟VNC Viewer，輸入localhost:1連線

  11. 從右上角應用程式按Run Program，搜尋winetricks --gui，執行後Wine會進行更新，再安裝mono，大約5分鐘。

  12. 如果Debian沒有聲音，請參考下方Bilibili文章安裝PulseAudio。

  13. 接著就可以來試試各種32位元Widnows程式了！
    如果想知道如何免Root存取Termux的內部目錄，請看這篇: 手機檔案管理器存取Termux的目錄

總結

跟Box86/Box64 + Wine比起來，雖然Exagear只能執行32位元的程式，但至少Termux本身是64位元的情況下可以直接裝32位元的Debian，不需要整個重裝32位元版的Termux。

Exagear唯一的缺點是閉源收費，導致網路上流傳的版本似乎都是盜版，包括這個Debian的版本，並且Proot的速度很慢。
