真．如何在手機上安裝Windows 11 (小米Pocophone F1)

發表於 2022-02-09 更新於 2022-02-11 分類於 電腦科學

Renegade Project這個專案主要是研究如何將Windows裝到手機上，不是虛擬機，而是真的把Android刷成Windows系統，有驅動的情況下還能玩電腦遊戲。

Github上有中國人釋出的原始碼，最關鍵的技術是把Linux kernel偽裝成UEFI，才能啟動Windows 11。目前處理器為高通s835、s845、s855的手機都有機會裝上ARM版的Windows 11。

本文先概覽小米Pocophone F1安裝Windows 11的執行效果，再來介紹如何安裝。

Poco F1裝Windows 11的效能

螢幕亮度、WIFI正常，有3D硬體加速；傳感器無法運作，沒有音效驅動。

ARM版的Office正常執行。

至於一般x86電腦的exe，Windows 11 ARM可轉譯執行x86的exe，此外有GPU硬體加速能玩3D遊戲。

開得起來的主要為32位元x86遊戲，而64位元x86就算能開效能也很差。

雖然微軟2021年出了OpenGL的相容層，但64位元exe僅是能開，談不上硬體加速。

接下來看看如何安裝至手機。以下步驟主要參考這部影片；他的頻道還有刷回Android的教學。

步驟簡介: 解鎖Bootloader → 刪除手機全部分區 → 建立 Windows用的分區 → 用特製版Windows PE開機，安裝Windows和驅動程式。

硬體資訊

  • 電腦: Windows 11 x86_64
  • 手機: 小米Pocophone F1，原廠MIUI(Android 10)，已解鎖Bootloader。
  • USB Type-C的擴充座(或OTG)，用於給手機連接實體鍵盤和滑鼠用。
  • 64GB隨身碟

事前準備

解鎖手機Bootloader鎖: 插入SIM卡綁定手機，到小米官網申請解鎖

然後下載以下檔案 (驅動程式最新版本可到Renegade Project官網下載)

  • Windows 11 ARM版的ISO: 用UUP DUMP下載。
  • Platform Tools (ADB & Fastboot)，解壓縮後放到C槽任意位置並加入至PATH環境變數。
  • Poco F1的TWRP: 下載最新版
  • UEFI開機映像:下載boot-beryllium.img
  • 驅動程式:下載beryllium.tar.gz，解壓縮後將目錄重新命名為Drivers。
  • Windows PE
  • DISM++，解壓縮後重新命名為DISM。
  • PARTED執行檔
  • commands.txt

  2. 進入Drivers目錄，會有一個output目錄。

  3. 將Win11 ISO、DISM、commands.txt、output四個項目複製到隨身碟。

開始安裝

  1. 手機關機，長按開機+音量下鍵進入fastboot模式，連上電腦。在Windows11ARM目錄右鍵，在此開啟終端機，輸入指令用TWRP開機 (注意: 此TWRP並沒有安裝到手機上)。

  3. 進入手機的adb shell

  4. 將parted檔複製到sbin目錄，並修改權限

  6. 執行parted，進行硬碟分割

  27. 重開機後大概等個10分鐘，就會進入Windows 11的初始設定畫面了，WIFI正常所以不用擔心網路問題。

  28. 需注意的是不要讓手機螢幕熄滅，因為關閉螢幕後會無法喚醒。另外因為傳感器無法使用，要到系統設定手動旋轉螢幕。關機的話就點選Windows的關機按鈕。

總結

選這支的理由是拍賣剛好看有很便宜的二手現貨，而且Poco F1刷機資源也很多，除了Android ROM以外，還有postMarketOS和Ubuntu touch能刷，且硬體支援度佳，不用屈就於Windows。

目前還有很多問題要解決(Renegade Project的表格寫不支援音效)，不然其實當新時代Windows Phone還蠻好玩的。
