Ubuntu Touch通用安裝教學

發表於 2022-02-13 分類於 電腦科學

Ubuntu Touch是手機Linux行動作業系統，原本是由Ubuntu母公司Canonical開發，現由UBports基金會維護。支援的手機相較postmarketOS來說比較少。

介面類似Ubuntu 16.04，有些機型會使用Halium Project的相容層解決硬體問題。

為簡化安裝流程，凡是有官方支援的機型(列表點此)，皆可透過官方提供的安裝器刷入手機。

相較之下，如果該機型只有unofficial port(非官方移植)的，就要手動刷入作者提供的原廠Android韌體和Ubuntu Touch刷機包。

下面例子，手機為官方支援的小米Pocophone F1，電腦為Ubuntu 20.04。

電腦安裝UBports Installer

到官網取得安裝包，或者Ubuntu直接用Snap安裝:

雖然有Windows版的，不過Linux的會比較穩定點，Windows版可能會抓不到手機。

刷入Ubuntu Touch

  1. 手機重開機至fastboot，連到電腦。

  2. UBports Installer應會自動識別裝置，之後按照螢幕操作，就會開始下載安裝包。

  3. 刷入之後可能還要手動重開機進Recovery，繼續按照螢幕指示，等它跑完。

  4. 之後手機重開機，安裝就完成了。

後續安裝優化

Ubuntu Touch操作邏輯為右側螢幕往內滑進入多工，左側螢幕往內滑開啟應用程式選單，螢幕從上面往下拉開啟狀態欄。

  1. Ubuntu Touch內建常用軟體，例如瀏覽器、相機、檔案管理器、終端機。

  2. 注意system分區預設是唯讀的，因此要到OpenStore商店下載APP；系統則是採OTA更新。

  3. 若要用APT安裝套件，以下的指令可以解除系統唯讀，但是隨意apt upgrade系統套件很容易導致手機卡開機。

  4. 如果要使用Linux桌面軟體，先安裝UT Tweak Tool開啟「桌面模式」，再用Libertine建立容器執行。

  5. 如果要執行Android APP，請安裝Waydroid。

  6. 手機連到電腦用MTP傳輸檔案。
